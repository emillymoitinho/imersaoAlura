📝  Nesta aula aprendi sobre estes conteúdos:

- Compreender as diferenças entre HTML, CSS e JavaScript;
- Fazer a sidebar navigation;

- Introduzir as propriedades de estilo do CSS no projeto, com truques no Devtools;
